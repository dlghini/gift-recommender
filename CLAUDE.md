# Project Rules

- Always use Next.js App Router conventions and TypeScript.
- Always build components using Tailwind CSS and shadcn/ui.
- Maintain a `MEMORY.md` file in the root directory. Before starting any new task, read `MEMORY.md` to gain context. After completing a task, update `MEMORY.md` with the new state of the app.
- After successfully completing a feature and updating `MEMORY.md`, always run `npm run build` to check for type errors, then commit the changes to Git.
